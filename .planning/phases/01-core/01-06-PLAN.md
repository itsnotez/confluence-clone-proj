frontend/src/api/content.js:
```js
import api from './axios'
export default {
  getContentTree: (spaceKey) => api.get(`/spaces/${spaceKey}/contents`),
  createContent: (spaceKey, data) => api.post(`/spaces/${spaceKey}/contents`, data),
  getContent: (id) => api.get(`/contents/${id}`),
  updateContent: (id, data) => api.put(`/contents/${id}`, data),
  publishContent: (id, data) => api.post(`/contents/${id}/publish`, data),
  deleteContent: (id) => api.delete(`/contents/${id}`),
  getVersions: (id) => api.get(`/contents/${id}/versions`)
}
```

frontend/src/stores/content.js (Pinia Composition API):
- currentContent: ref(null) — 현재 보고 있는 콘텐츠
- versions: ref([]) — 버전 목록
- isEditing: ref(false)
- fetchContent(id): contentApi.getContent(id) → currentContent.value = data.data
- saveContent(id, body): contentApi.updateContent(id, { body }) — Draft 저장
- publishContent(id, body): contentApi.publishContent(id, { body }) → currentContent 업데이트
- createContent(spaceKey, data): contentApi.createContent(spaceKey, data) → 생성된 id 반환
- fetchVersions(id): contentApi.getVersions(id) → versions.value

TipTapEditor.vue:
- props: modelValue(String — TipTap JSON), readonly(Boolean 기본 false)
- emits: ['update:modelValue']
- useEditor: StarterKit, Image, Link, Table, CodeBlockLowlight 확장 포함
- readonly=true 이면 editable: false (읽기 모드)
- 에디터 콘텐츠 초기화: JSON.parse(modelValue) → editor.commands.setContent()
- 에디터 변경시: JSON.stringify(editor.getJSON()) → emit('update:modelValue', json)
- 툴바 (readonly=false 시에만 표시): Bold, Italic, Underline, H1/H2, BulletList, OrderedList, Code, Image 삽입, Link
  - DevExtreme DxToolbar 또는 직접 버튼 HTML 사용
- EditorContent 컴포넌트로 에디터 렌더링
- onBeforeUnmount: editor.destroy()
- 스타일: 에디터 영역 min-height 400px, border 1px solid #ddd, padding 16px
