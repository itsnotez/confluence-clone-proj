frontend/src/api/space.js:
```js
import api from './axios'
export default {
  getSpaces: () => api.get('/spaces'),
  getSpace: (spaceKey) => api.get(`/spaces/${spaceKey}`),
  createSpace: (data) => api.post('/spaces', data),
  updateSpace: (spaceKey, data) => api.put(`/spaces/${spaceKey}`, data),
  deleteSpace: (spaceKey) => api.delete(`/spaces/${spaceKey}`),
  toggleFavorite: (spaceKey) => api.post(`/spaces/${spaceKey}/favorite`),
  getContentTree: (spaceKey) => api.get(`/spaces/${spaceKey}/contents`)
}
```

frontend/src/stores/space.js (Pinia Composition API):
- spaces: ref([]) — Space 목록
- currentSpace: ref(null) — 현재 선택된 Space
- contentTree: ref([]) — 현재 Space의 콘텐츠 트리
- fetchSpaces(): spaceApi.getSpaces() → spaces.value = data.data
- fetchSpace(spaceKey): spaceApi.getSpace(spaceKey) → currentSpace.value = data.data
- createSpace(formData): spaceApi.createSpace(formData) → fetchSpaces()
- deleteSpace(spaceKey): spaceApi.deleteSpace(spaceKey) → fetchSpaces()
- toggleFavorite(spaceKey): spaceApi.toggleFavorite(spaceKey) → 해당 space.favorited 토글
- fetchContentTree(spaceKey): spaceApi.getContentTree(spaceKey) → contentTree.value = data.data
  </action>
